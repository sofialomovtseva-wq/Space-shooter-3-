# Space-shooter-3-
Space Shooter — это 2D-космический шутер, в котором игрок управляет кораблём, сражается с астероидами и врагами, уклоняется от препятствий и зарабатывает ресурсы. Во время полётов играет атмосферная музыка, а после прибытия на станции появляется возможность торговать с NPC, улучшать корабль и готовиться к новым вылетам.
